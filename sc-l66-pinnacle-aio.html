Description

Information about the data that is written to the data packet
  • Sport: Soccer
  • Bookmaker: Pinnacle
  • Leagues: L-66
  • Seasons: 2008/2009 - 2019/2020
  • Type of odds lines: Opening and Closing
  • Odds lines: 1X2, Double Chance, Total Goals, Asian Handicap, Both team to Score
  • Event dates: Shows in GMT (UTC +0)
  • Basic event data: Country, League, Season, Teams, Leagues ID (FIFA)
  • Results: Final & 1st, 2nd Halfs
  • Extra Time Data: Extra Time scores, Awarded, Penalties
  • Odds lines: Structured by blocks and sub-blocks
  • Additional settings: Payouts, Movements
